# 🚀 Wendel LoadScreen

Uma loadscreen moderna, otimizada e totalmente configurável para **FiveM**, desenvolvida do zero com foco em performance, experiência do
usuário e facilidade de personalização.
------------------------------------------------------------------------

## ✨ Recursos

-   🎨 Interface moderna e minimalista
-   ⚡ Desenvolvida do zero
-   🔥 Compatível com QBOX
-   📱 Layout responsivo (16:9, ultrawide e resoluções menores)
-   🌴 Animações suaves no cenário
    -   Movimento leve dos coqueiros simulando vento
    -   Zoom cinematográfico contínuo
    -   Movimento sutil dos personagens
-   📊 Barra de progresso sincronizada com o carregamento dos recursos
-   📂 Exibição do recurso sendo carregado:

``` text
phoenix/assets/nome_do_recurso
```

-   🎵 Sistema de música local (.mp3)
-   🔊 Controle de volume
-   ⏯️ Play/Pause
-   🔁 Loop automático
-   🌐 Redes sociais configuráveis
-   🔗 Botões clicáveis
-   ✨ Fade In / Fade Out
-   🖥️ Totalmente otimizada para FiveM NUI
-   ⚙️ Configuração centralizada

------------------------------------------------------------------------

# 🎵 Sistema de Música

O sistema utiliza apenas **MP3 local**, garantindo máxima
compatibilidade e eliminando problemas comuns relacionados ao YouTube ou
serviços externos.

Basta substituir:

``` text
html/assets/audio/loading.mp3
```

Não é necessário alterar nenhum código.

------------------------------------------------------------------------

# ⚙️ Configuração

Toda a personalização pode ser realizada através do arquivo:

``` text
config.lua
```

É possível alterar:

-   Nome do servidor
-   Cor da barra de progresso
-   Links das redes sociais
-   Música
-   Volume
-   Logo
-   Background
-   Velocidade das animações
-   Intensidade do zoom
-   Movimento dos personagens
-   Movimento dos coqueiros

Sem necessidade de editar HTML ou JavaScript.
